  2. I found this github repo that seems quite relevant, so I decided to give it a try

  3. I downloaded and installed MySQL and got a server running.

  4. I followed the instructions from the repository. I was quite clueless at some of them, namely:

    but I figured them out by reading the source code and searching on google.
